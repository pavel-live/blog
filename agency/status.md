## Статус работы агентов по задачам блога

Используй этот файл как «текстовую доску» для отслеживания прогресса.

### Формат

```markdown
## Задача: Краткое название фичи

- ID: blog_feature_tags
- Статус: new | in_progress | paused | done | cancelled
- Ответственный (человек): (опционально)

### Этапы
- [ ] ТЗ (аналитик)
- [ ] Review ТЗ
- [ ] Архитектура
- [ ] Review архитектуры
- [ ] План
- [ ] Review плана
- [ ] Разработка (все задачи выполнены)
- [ ] Review кода

### Ссылки на артефакты
- ТЗ: `agency/tz/...`
- Архитектура: `agency/architecture/...`
- План: `agency/plans/...`
- Задачи: `agency/task_descriptions/...`
- Отчёты о тестах: `agency/test_reports/...`

### Открытые вопросы по задаче
- См. `agency/open_questions.md` (фильтруй по ID задачи).
```

### Пример записи

```markdown
## Задача: Страница тегов

- ID: blog_feature_tags
- Статус: in_progress

### Этапы
- [x] ТЗ (аналитик)
- [x] Review ТЗ
- [ ] Архитектура
- [ ] Review архитектуры
- [ ] План
- [ ] Review плана
- [ ] Разработка
- [ ] Review кода

### Ссылки на артефакты
- ТЗ: `agency/tz/blog_feature_tags_TZ.md`
```
