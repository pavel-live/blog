## 09_agent_code_reviewer (ревьюер кода)

Полный промпт: `../agents/09_agent_code_reviewer.md`

Роль: проверить качество кода, соответствие ТЗ и плана, а также результаты тестирования.

### Что давать ревьюеру кода

- Изменённые файлы / дифф коммита.
- Описание задачи:  
  `agency/task_descriptions/...`
- ТЗ и, при необходимости, архитектуру.
- Отчёт о тестировании из `agency/test_reports/...`.

### Что ожидать

- Список конкретных замечаний по коду и тестам.
- Рекомендации, какие места нужно доработать.

### Пример текстового запроса

```text
Ты работаешь по промпту из ../agents/09_agent_code_reviewer.md.

Проверь реализованную задачу (описана в agency/task_descriptions/task_1_2_blog_feature_tags_example.md)
на соответствие ТЗ и плану, а также отчёт о тестировании
из agency/test_reports/test_report_task_1_2_blog_feature_tags_example.md.

Дай список конкретных замечаний по коду и тестам, если они есть.
```
