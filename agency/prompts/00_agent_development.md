## 00_agent_development (общее описание мультиагентной разработки)

Полная версия промпта находится в соседнем репозитории:  
`../agents/00_agent_development.md`

Этот файл используется как **общая концепция**: он описывает роли агентов, этапы (анализ → архитектура → план → разработка), лимиты итераций и правила взаимодействия.

### Как применять в блоге

- При запуске большой фичи (новый раздел блога, крупное изменение структуры) можно:
  1. Подгрузить текст `../agents/00_agent_development.md` как системный контекст.
  2. Дополнить его ссылками на:
     - `agency/PROJECT_OVERVIEW.md`
     - `agency/ARCHITECTURE.md`
     - файл задачи из `agency/tasks/`.

### Пример команды (псевдокод)

```bash
cursor-agent -f --model auto \
  -p "$(cat ../agents/00_agent_development.md)" \
  -p \"Описание задачи: см. agency/tasks/blog_feature_tags_example.md\"
```
