## Открытые вопросы от агентов

Любой агент, у которого возникли блокирующие вопросы, должен фиксировать их здесь.

### Формат записи

```markdown
## Задача: Краткое название фичи

- ID задачи: blog_feature_tags
- Дата: 2026-02-23
- Агент: Аналитик | Архитектор | Планировщик | Разработчик | Ревьюер ТЗ | Ревьюер архитектуры | Ревьюер плана | Ревьюер кода

### Вопросы
1. Вопрос 1?
2. Вопрос 2?

### Контекст
- Ссылка на ТЗ/план/код, к которому относится вопрос.
```

### Пример записи

```markdown
## Задача: Страница тегов

- ID задачи: blog_feature_tags
- Дата: 2026-02-23
- Агент: Аналитик

### Вопросы
1. Нужно ли поддерживать страницу тега для каждого языка отдельно?
2. Какой максимальный размер списка постов на странице тега (пагинация)?

### Контекст
- Черновик ТЗ: `agency/tz/blog_feature_tags_TZ_draft.md`
```
